# v1.0.0

Release date: 2025-12-20

## Highlights

- First stable release of Kafscale.
- Kafka protocol compatibility for core produce/fetch, consumer groups, and admin ops.
- Stateless brokers with S3-backed durability and etcd-backed metadata/offsets.
- Operator-based Kubernetes deployment with console UI and Prometheus metrics.

## Known limitations

- TLS/SASL authentication is planned for a future release.
- No Kafka transactions, KRaft APIs, or embedded stream processing.

## Security fixes

- No known runtime vulnerabilities were fixed in this release.
